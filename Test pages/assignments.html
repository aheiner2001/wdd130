In this example, we're using the document.cookie API to store the checkbox state in a cookie called myCheckboxState. When the page loads, we check if the cookie value is true and set the checkbox's checked attribute accordingly. We also listen for changes to the checkbox state and update the cookie value accordingly. Note that this is just one way to keep a checkbox checked after a page refresh using cookies. You could also use local storage or session storage to achieve the same effect.
